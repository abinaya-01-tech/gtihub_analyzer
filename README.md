# 🚀 GitHub Profile Analyzer

A web application that analyzes GitHub profiles and presents valuable insights through an interactive dashboard. The application fetches real-time data from the GitHub API and visualizes repository statistics, programming language usage, followers, and other profile metrics using dynamic charts.

---

## 📖 Overview

GitHub Profile Analyzer is a Flask-based web application designed to help users explore and understand GitHub profiles. By simply entering a GitHub username, users can view detailed information about repositories, top programming languages, follower statistics, and other profile insights in an easy-to-read dashboard.

This project demonstrates API integration, backend development, data visualization, and responsive web design.

---

## ✨ Features

- 🔍 Search any public GitHub profile
- 👤 Display user profile information
- 📂 View public repository statistics
- 💻 Analyze programming language distribution
- ⭐ Display total stars across repositories
- 👥 Show followers and following count
- 📊 Interactive charts using Chart.js
- 📱 Responsive and user-friendly interface
- ⚡ Real-time data fetched using the GitHub API

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/abinaya-01-tech/github-profile-analyzer.git
```

### 2. Navigate to the Project Folder

```bash
cd github-profile-analyzer
```

### 3. Install Required Packages

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
python app.py
```

### 5. Open Your Browser

```
http://127.0.0.1:5000
```
